## 🚀 OVERVIEW
This is my personal professional repository. It's a highly interactive, cyberpunk-themed portfolio built to showcase my technical pivot from Customer Operations to **Workflow Automation** and **Algorithmic Trading (XAUUSD)**.

## 🛠️ TECH STACK & ARCHITECTURE
*   **Frontend:** HTML5, Tailwind CSS (CDN), Custom CSS Animations.
*   **Logic:** Vanilla JavaScript (IntersectionObserver, DOM Manipulation).
*   **Aesthetics:** Deep Obsidian Black & Electric Cyan Neon Theme.

## 💎 CORE FEATURES (THE "RARE" ELEMENTS)
*   **Matrix Decryption Effect:** Dynamic text scramble on load for hero subtitles.
*   **Interactive Resume Modal:** Integrated video background (`assets/Resume vid.mp4`) with glassmorphism UI and auto-download PDF feature.
*   **Floating Global Header:** Mid-top navigation bar providing real-time status and contact info without obstructing page flow.
*   **Animated Core Skills:** Progress bars that animate based on user scroll position.
